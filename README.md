# Crust & Cream - Premium Fast Food Restaurant

**Crust & Cream** is a modern, responsive web application for a premium online fast food restaurant. We specialize in deliciously crafted pizzas, juicy burgers, creamy pastas, and golden fries, blending bold flavors with a luxurious dining experience.

## Project Overview

Crust & Cream is designed to provide a seamless user experience for food enthusiasts. The website serves as a digital storefront where users can explore the menu, view special offers, read customer testimonials, and book a table online.

> **Tagline:** Where crispy crust meets creamy perfection.

---

## Features

### 1. Interactive Home Page

* **Hero Slider:** A dynamic carousel showcasing the restaurant's commitment to speed, freshness, and quality ingredients.
* **Special Offers:** Highlighting promotional "Tasty Thursdays" and "Pizza Days" with discount badges.

### 2. Digital Menu

* **Dynamic Filtering:** An Isotope-powered menu that allows users to filter items by category: Burgers, Pizzas, Pasta, and Fries.
* **Detailed Product Cards:** Each item features a high-quality image, price, and description.

### 3. Online Table Booking

* A dedicated booking section featuring a functional reservation form (Name, Phone, Email, Person Count, and Date).
* **Integrated Google Maps:** Provides a visual location of the restaurant for easy navigation.

### 4. About & Testimonials

* A dedicated section detailing the brand story.
* **Customer Reviews:** A slider featuring feedback from satisfied clients.

### 5. Responsive Design

* Fully optimized for mobile, tablet, and desktop screens using Bootstrap 4 and custom media queries.

---

## Technologies Used

### Frontend

* **HTML5 & CSS3:** For semantic structure and advanced styling.
* **Bootstrap 4:** For the responsive grid system and pre-styled components.
* **JavaScript (jQuery):** For DOM manipulation and interactivity.

### Plugins & Libraries

* **Owl Carousel:** For the testimonial and hero sliders.
* **Isotope:** For the filterable food menu.
* **Font Awesome:** For social media and UI icons.
* **Nice Select:** For styled and user-friendly dropdown menus.
* **Google Maps API:** For location services.


## Installation & Usage

1. **Clone the repository:**
```bash
git clone https://github.com/anaskhan-codelab/crust-and-cream

```


2. **Open the project:**
Navigate to the project folder and open `index.html` in any modern web browser.
3. **Customization:**
* To change the menu items, edit the `.grid` section in `menu.html`.
* To update contact info, modify the `footer` section across all HTML files.
